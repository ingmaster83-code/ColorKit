🎨 컬러킷이란?

컬러킷(ColorKit)은 디자이너, 개발자, 크리에이터를 위한 무료 온라인 색상 도구 모음입니다.

HEX·RGB·HSL 색상 변환, 이미지 색상 추출, 그라디언트 및 팔레트 생성, WCAG 접근성 대비 검사, CSS 박스·텍스트 그림자 코드 생성까지 색상 작업에 필요한 모든 도구를 한 곳에서 무료로 제공합니다.

✅ 서비스 특징

  • HEX, RGB, HSL 세 가지 색상 코드 상호 변환
  • 이미지에서 색상 직접 추출 (캔버스 픽셀 샘플링)
  • CSS 그라디언트 코드 즉시 생성
  • 보색·유사색·트라이어드 팔레트 생성
  • WCAG AA·AAA 접근성 대비율 검사
  • CSS box-shadow / text-shadow 코드 생성기
  • 불투명도 변환 (rgba / 8자리 HEX / hsla)
  • 모든 처리가 브라우저 안에서 완결 — 서버로 데이터 전송 없음
  • 회원가입 없이 무제한 무료 사용
  • PC·태블릿·스마트폰 반응형 지원
